### НАЧАЛО ОПИСАНИЯ ###
# slide: номер в очереди показа
# file: имя файла-картинки для слайда
# title: наименование (большим шрифтом)
# desc: краткое описание (стандартным)
# url: ссылка на страницу (через кнопку Подробнее)
# text-position: позиция по вертикали: top, center, bottom (рассчитывается относительно высоты слайда, применяется для title; desc располагается под title; url под desc)
# text-align: позиция по горизонтали: left, center, right (рассчитывается относительно ширины экрана, применяется для title, desc, url)
# duration: продолжительность в секундах, в течение которого показывать слайд
# title-effects: эффекты для title (например, zoom, fade-in, fade-out и другие)
# desc-effects: эффекты для desc (например, zoom, fade-in, fade-out и другие)
# url-effects: эффекты для url (например, zoom, fade-in, fade-out и другие)
# slide-effect: эффект для slide (например, zoom, fade-in, fade-out и другие)
